# CaptureCA

This Magisk/KernelSU module safely moves user certificates generated by PCAPdroid (or similar proxy/network monitoring tools) to the system level (System Store).

It allows you to bypass the persistent **"Network may be monitored"** warning on your device and enables you to analyze HTTPS traffic seamlessly.

## 🚀 Features
*   **Full System Integration:** Injects user certificates directly into the `system/etc/security/cacerts/` directory.
*   **Clean Installation:** Performs operations systemlessly via the `post-fs-data.sh` script, without physically modifying system partitions.
*   **Compatibility:** Works flawlessly with Magisk and KernelSU supported devices.

## 📂 Module Structure
This module follows the standard Magisk module hierarchy:
*   `module.prop`
*   `customize.sh`
*   `post-fs-data.sh`
*   `system/etc/security/cacerts/` (Certificate files are located here)

## 🛠️ Installation
1.  Download the latest `CaptureCA.zip` release from this repository.
2.  Open the Magisk Manager or KernelSU application.
3.  Navigate to the **Modules** tab.
4.  Tap on **"Install from storage"** and select the downloaded `CaptureCA.zip` file.
5.  Reboot your device once the installation is complete.
